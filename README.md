# 📚 Library Management Database

## 📖 Project Overview
This project is a beginner-friendly Library Management Database built using MySQL. It demonstrates database design, table relationships, CRUD operations, joins, and subqueries.

## 🚀 Features
- Manage librarian records
- Manage member records
- Manage book records
- Issue and return books
- Track issued books
- Perform SQL queries for data retrieval

## 🛠 Technologies Used
- MySQL
- MySQL Workbench
- SQL

## 📂 Database Tables
- Librarian
- Members
- Books
- Issue_Book

## 📚 SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT
- SELECT
- WHERE
- ORDER BY
- COUNT()
- MAX()
- MIN()
- UPDATE
- DELETE
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- SUBQUERY
